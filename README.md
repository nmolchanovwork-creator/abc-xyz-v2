# Интегральный ABC-XYZ анализ

## Установка и запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Как начать работу

1. В боковой панели скачайте шаблон Excel («📥 Скачать шаблон Excel»).
2. Заполните его своими данными о продажах (по месяцам, годам, категориям и т.д.).
3. Загрузите заполненный файл через «Загрузить файл» в боковой панели.
4. Либо выберите «Демо-данные», чтобы сразу посмотреть, как работает приложение.

## Структура файла с данными

Обязательные колонки:

| Колонка | Обязательна | Описание |
|---|---|---|
| SKU | Да | Уникальный код товара |
| Наименование | Да | Название товара |
| Категория | Да | Категория (фильтр) |
| Тип продажи | Да | розница / опт / индивидуально (фильтр) |
| Месяц | Да | Название месяца по-русски |
| Год | Да | Год числом |
| Себестоимость | Да | Число |
| Цена | Да | Число |
| Продажи шт. | Да | Количество продаж за месяц |
| Текущий остаток | Да | Остаток на складе |
| Выручка | Нет | Если пусто — считается как Продажи шт. × Цена |
| Маржа | Нет | Если пусто — считается как Продажи шт. × (Цена − Себестоимость) |

Важно: одна строка = один SKU за один месяц. Если товар продавался несколько
месяцев — для него должно быть несколько строк (нужно для расчёта XYZ).

Точный образец — в скачиваемом шаблоне (лист «Данные» + лист «Инструкция»).

## ИИ-модуль: OpenAI и Gemini

В боковой панели выбирается провайдер — **OpenAI** или **Gemini** — и модель:

- OpenAI: gpt-4o-mini, gpt-4o, gpt-4.1-mini, gpt-4.1
- Gemini: gemini-2.0-flash, gemini-2.0-flash-lite, gemini-1.5-flash, gemini-1.5-pro

API-ключ вводится прямо в интерфейсе и не сохраняется — используется только
в рамках текущей сессии браузера.

AI-советы по SKU запрашиваются по кнопке (с ограничением количества строк),
а не автоматически на каждую строку — чтобы контролировать расход запросов.
Результаты кэшируются в сессии.

## Экспорт результатов

- **Сводная таблица** — кнопка «📥 Скачать сводную таблицу (Excel)» на странице
  «Аналитика и Действия».
- **История AI-стратегий** — все сгенерированные за сессию стратегии сохраняются
  на странице «Глобальная ИИ-Стратегия», их можно посмотреть в списке или
  скачать всё разом кнопкой «📥 Скачать всю историю (.txt)».

## Порядок букв в ABC-коде

`ABC_код` строится в фиксированном порядке: **Выручка → Количество → Маржа**.
Например, `CAA` означает: C по выручке, A по количеству, A по марже.

## Цветовая подсветка таблицы

- **ABC** (Продажи шт. / Выручка / Маржа — каждая колонка по своей букве): 🟢 A · 🟡 B · 🔴 C
- **XYZ** (стабильность спроса, отдельная палитра): 🔵 X · 🟣 Y · 🟠 Z

Если выбрана только одна категория в фильтре — колонка «Категория» в таблице скрывается автоматически.

## Вкладка «Рекомендации»

Готовые (заранее заготовленные) правила по всем 27 комбинациям ABC-кода — файл
`utils/recommendations.csv`. Можно редактировать этот файл напрямую (тот же формат:
колонка `Категория` с кодом AAA...CCC + правила по цене/скидке/остатку/продвижению/выводу),
изменения подхватятся при перезапуске приложения.

## Дашборды

- KPI-показатели: суммарная выручка/маржа, рентабельность, число SKU, число кандидатов
  на делистинг, оценка капитала, замороженного в неликвиде (XYZ = Z).
- Парето-диаграмма — концентрация выручки по SKU (суть ABC-анализа).
- Распределение маржи по ABC и распределение SKU по XYZ.
- Матрица ABC × XYZ.
- Оборачиваемость запаса по ABC-группам (продажи / остаток).
- Динамика продаж топ-товаров.
